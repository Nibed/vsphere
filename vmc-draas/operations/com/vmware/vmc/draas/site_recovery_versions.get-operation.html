    vmc draas site recovery versions: get

    Query site recovery versions for the specified sddc

    Request:

    HTTP request

    GET https://{server}/rest/vmc/draas/api/orgs/{org}/sddcs/{sddc}/site-recovery/versions

    ?version_source=string

    Path Parameters

    Name Type Description
    Required
    org string Organization identifier (required).
    sddc string sddc identifier (required).

    Request Query Parameters:

    Name Type Description
    bold = required
    version_source string Optional. Represents the source for getting the version from. (optional).

    Response:

    HTTP Status Code: 200

    Representation:
