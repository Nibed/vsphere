    vmc model: feature capabilities

    List of features and their capability details based on NSX Edge appliance form factor.

    Representation:

    • JSON
    {
        "feature_capabilities" : [
            {
                "is_supported" : true,
                "service" : "string",
                "configuration_limits" : [
                    {
                        "value" : "string",
                        "key" : "string"
                    },
                    {
                        "value" : "string",
                        "key" : "string"
                    }
                ],
                "permission" : {
                    "data_permission" : {
                        "save_permission" : true,
                        "publish_permission" : true
                    },
                    "access_permission" : {
                        "manage_permission" : true,

                    "is_licensed" : true
                }
            },
            {

    Attributes:

    Name Type Description
    Optional
    timestamp long Time stamp value at which the feature capabilities were retrieved. format: int64

    Optional.

    feature_capabilities feature_capability[] List of feature capability information.

    Optional.

    feature_capabilities[].configuration_limits key_value_attributes[] List of key value pairs describing the feature configuration limits.

    Optional.

    feature_capabilities[].configuration_limits[].value string Value corresponding to the key of the configuration limit parameter.

    Optional.

    feature_capabilities[].configuration_limits[].key string Key name of the configuration limit parameter.

    Optional.

    feature_capabilities[].is_supported boolean Value is true if feature is supported on NSX Edge.

    Optional.

    feature_capabilities[].service string Name of the feature or service.

    Optional.

    feature_capabilities[].permission licence_acl_permissions Licence and access control information for the feature.

    Optional.

    feature_capabilities[].permission.data_permission data_permissions Data access control information for the feature.

    Optional.

    feature_capabilities[].permission.data_permission.save_permission boolean Optional.
    feature_capabilities[].permission.data_permission.publish_permission boolean Optional.
    feature_capabilities[].permission.is_licensed boolean Value is true if feature is licenced.

    Optional.

    feature_capabilities[].permission.access_permission interaction_permissions Access control information for the feature.

    Optional.

    feature_capabilities[].permission.access_permission.manage_permission boolean Optional.
