    Optional.

    enable_send_redirects boolean Value is true if send redirects is enabled. Enable ICMP redirect to convey routing information to hosts.

    Optional.

    in_shaping_policy.burst_size long Optional.

    label string Interface label of format vNic_{vnicIndex} provided by NSX Manager. Read only.

    Optional.

    enable_proxy_arp boolean Value is true if proxy arp is enabled. Enable proxy ARP if you want to allow the NSX Edge of type 'gatewayServices' to answer ARP requests intended for other machines.

    Optional.

    name string Name of the interface. Optional.

    Optional.

    mtu long MTU of the interface, with default as 1500. Min is 68, Max is 9000. Optional. format: int32

    Optional.

    fence_parameters key_value_attributes[] Optional.
    fence_parameters[].value string Value corresponding to the key of the configuration limit parameter.

    Optional.

    fence_parameters[].key string Key name of the configuration limit parameter.

    Optional.

    out_shaping_policy.enabled boolean Optional.

    portgroup_name string Name of the port group or logical switch.

    Optional.

    enable_bridge_mode boolean Value is true if bridge mode is enabled.

    Optional.

    type string Type of the vnic. Values are uplink, internal, trunk. At least one internal interface must be configured for NSX Edge HA to work.

    Optional.

    portgroup_id string Value are port group ID (standard portgroup or distributed portgroup) or virtual wire ID (logical switch). Logical switch cannot be used for a TRUNK vnic. Example 'network-17' (standard portgroup), 'dvportgroup-34' (distributed portgroup) or 'virtualwire-2' (logical switch).
