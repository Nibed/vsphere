    vmc orgs subscriptions offer instances: list

    List all offers available for the specific product type in the specific region

    Request:

    HTTP request

    GET https://{server}/rest/vmc/api/orgs/{org}/subscriptions/offer-instances

    ?product=string
    &product_type=string
    &region=string
    &type=string

    Path Parameters

    Name Type Description
    Required
    org string Organization identifier. (required).

    Request Query Parameters:

    Name Type Description
    bold = required
    region string Region for the offer (required).
    product_type string Type of the product in offers. *This has been deprecated*. Please use product & type parameters (required).
    product string Optional. The product that you are trying to purchase, eg. host. This needs to be accompanied by the type parameter (optional).
    type string Optional. The type/flavor of the product you are trying it purchase,eg. an `r5.metal` host. This needs to be accompanied by the product parameter. (optional).
