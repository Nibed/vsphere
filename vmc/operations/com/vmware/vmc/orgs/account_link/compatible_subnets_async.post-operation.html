    vmc orgs account link compatible subnets async: post

    Sets which subnet to use to link accounts and finishes the linking process via a task

    Request:

    HTTP request

    POST https://{server}/rest/vmc/api/orgs/{org}/account-link/compatible-subnets-async

    Path Parameters

    Name Type Description
    Required
    org string Organization identifier. (required).

    Request Body Structure:

    {
        "aws_subnet" : {
            "availability_zone" : "string",
            "vpc_id" : "string",
            "connected_account_id" : "string",
            "name" : "string",
            "subnet_id" : "string",
            "is_compatible" : true,
            "region_name" : "string",
            "subnet_cidr_block" : "string",
            "vpc_cidr_block" : "string"
        }
    }

    Request Body Parameters:

    Name Type Description
    bold = required
    aws_subnet aws_subnet The subnet chosen by the customer (required).
    aws_subnet.connected_account_id string The connected account ID this subnet is accessible through. This is an internal ID formatted as a UUID specific to Skyscraper.

    Optional.

    aws_subnet.region_name string The region this subnet is in, usually in the form of country code, general location, and a number (ex. us-west-2).

    Optional.

    aws_subnet.availability_zone string The availability zone this subnet is in, which should be the region name plus one extra letter (ex. us-west-2a).

    Optional.

    aws_subnet.subnet_id string The subnet ID in AWS, provided in the form 'subnet-######'.

    Optional.

    aws_subnet.subnet_cidr_block string The CIDR block of the subnet, in the form of '#.#.#.#/#'.

    Optional.

    aws_subnet.is_compatible boolean Flag indicating whether this subnet is compatible. If true, this is a valid choice for the customer to deploy a SDDC in.

    Optional.

    aws_subnet.vpc_id string The VPC ID the subnet resides in within AWS. Tends to be 'vpc-#######'.

    Optional.

    aws_subnet.vpc_cidr_block string The CIDR block of the VPC, in the form of '#.#.#.#/#'.

    Optional.

    aws_subnet.name string Optional field (may not be provided by AWS), indicates the found name tag for the subnet.

    Optional.
