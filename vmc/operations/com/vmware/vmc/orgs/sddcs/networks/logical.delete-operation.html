    vmc orgs sddcs networks logical: delete

    Delete a network in an SDDC.

    Request:

    HTTP request

    DELETE https://{server}/rest/vmc/api/orgs/{org}/sddcs/{sddc}/networks/4.0/sddc/networks/{network_id}

    Path Parameters

    Name Type Description
    Required
    org string Organization identifier. (required).
    sddc string Sddc Identifier. (required).
    network_id string Logical Network Identifier (required).

    Response:

    HTTP Status Code: 200

    None

    Errors:

    HTTP Status Code Type Description
    400 com.vmware.vapi.std.errors.InvalidRequest Bad request. Request object passed is invalid.
    403 com.vmware.vapi.std.errors.Unauthorized Forbidden. Authorization header not provided.
    404 com.vmware.vapi.std.errors.NotFound Not found. Requested object not found.
