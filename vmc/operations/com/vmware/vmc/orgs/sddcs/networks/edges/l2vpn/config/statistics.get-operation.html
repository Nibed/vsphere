    vmc orgs sddcs networks edges l2vpn config statistics: get

    Retrieve L2 VPN statistics for a compute gateway (NSX Edge).

    Request:

    HTTP request

    GET https://{server}/rest/vmc/api/orgs/{org}/sddcs/{sddc}/networks/4.0/edges/{edge_id}/l2vpn/config/statistics

    Path Parameters

    Name Type Description
    Required
    org string Organization identifier. (required).
    sddc string Sddc Identifier. (required).
    edge_id string Edge Identifier. (required).

    Response:

    HTTP Status Code: 200

    Representation:

    {
        "value" : {
            "site_stats" : [
                {
                    "dropped_tx_packets" : 1,
                    "last_updated_time" : 1,
                    "tunnel_status" : "string",
                    "name" : "string",
                    "dropped_rx_packets" : 1,
                    "encryption_algorithm" : "string",
                    "failure_message" : "string",
                    "established_date" : 1,
                    "tx_bytes_from_local_subnet" : 1,
                    "rx_bytes_on_local_subnet" : 1
                },
                {
                    "dropped_tx_packets" : 1,
                    "last_updated_time" : 1,
                    "tunnel_status" : "string",
                    "name" : "string",
                    "dropped_rx_packets" : 1,
                    "encryption_algorithm" : "string",
                    "failure_message" : "string",
                    "established_date" : 1,
                    "tx_bytes_from_local_subnet" : 1,
                    "rx_bytes_on_local_subnet" : 1
                }
            ],
            "time_stamp" : 1,
            "server_status" : "string"
        }
    }

    Response Type:

    Name Type Description
    bold = required
    value l2vpn_status_and_stats com.vmware.vmc.model.L2vpnStatusAndStats.
    value.time_stamp long Time stamp of statistics collection. format: int64

    Optional.

    value.server_status string Optional.
    value.site_stats l2vpn_stats[] List of statistics for each Site.

    Optional.

    value.site_stats[].tunnel_status string Status of the tunnel (UP/DOWN).

    Optional.

    value.site_stats[].established_date long Tunnel established date. format: int64

    Optional.

    value.site_stats[].name string User defined name of the site.

    Optional.

    value.site_stats[].dropped_rx_packets long Number of received packets dropped.

    Optional.

    value.site_stats[].encryption_algorithm string Cipher used in encryption.

    Optional.

    value.site_stats[].failure_message string Reason for the tunnel down.

    Optional.

    value.site_stats[].tx_bytes_from_local_subnet long Number of bytes transferred from local subnet.

    Optional.

    value.site_stats[].rx_bytes_on_local_subnet long Number of bytes received on the local subnet.

    Optional.

    value.site_stats[].dropped_tx_packets long Number of transferred packets dropped.

    Optional.

    value.site_stats[].last_updated_time long Time stamp of the statistics collection. format: int64

    Optional.

    Errors:

    HTTP Status Code Type Description
    400 com.vmware.vapi.std.errors.InvalidRequest Bad request. Request object passed is invalid.
    403 com.vmware.vapi.std.errors.Unauthorized Forbidden. Authorization header not provided.
    404 com.vmware.vapi.std.errors.NotFound Not found. Requested object not found.
