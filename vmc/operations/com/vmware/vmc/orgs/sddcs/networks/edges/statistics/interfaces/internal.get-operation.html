    vmc orgs sddcs networks edges statistics interfaces internal: get

    Retrieve internal interface statistics for a management or compute gateway (NSX Edge).

    Request:

    HTTP request

    GET https://{server}/rest/vmc/api/orgs/{org}/sddcs/{sddc}/networks/4.0/edges/{edge_id}/statistics/interfaces/internal

    ?start_time=1
    &end_time=1

    Path Parameters

    Name Type Description
    Required
    org string Organization identifier. (required).
    sddc string Sddc Identifier. (required).
    edge_id string Edge Identifier. (required).

    Request Query Parameters:

    Name Type Description
    bold = required
    start_time long Optional. Show statistics from this start time (in milliseconds since epoch). (optional).
    end_time long Optional. Show statistics until this end time (in milliseconds since epoch). (optional).

    Response:

    HTTP Status Code: 200
