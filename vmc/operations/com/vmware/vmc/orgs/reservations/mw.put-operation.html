    vmc orgs reservations mw: put

    update the maintenance window for this SDDC

    Request:

    HTTP request

    PUT https://{server}/rest/vmc/api/orgs/{org}/reservations/{reservation}/mw

    Path Parameters

    Name Type Description
    Required
    org string Organization identifier. (required).
    reservation string Reservation Identifier (required).

    Request Body Structure:

    {
        "window" : {
            "hour_of_day" : 1,
            "day_of_week" : "string"
        }
    }

    Request Body Parameters:

    Name Type Description
    bold = required
    window maintenance_window Maintenance Window (required).
    window.day_of_week string Possible values are:
    • DAY_OF_WEEK_SUNDAY
      • ,
    • DAY_OF_WEEK_MONDAY
      • ,
    • DAY_OF_WEEK_TUESDAY
      • ,
    • DAY_OF_WEEK_WEDNESDAY
      • ,
    • DAY_OF_WEEK_THURSDAY
      • ,
    • DAY_OF_WEEK_FRIDAY
      • ,
    • DAY_OF_WEEK_SATURDAY
      • ,

    Optional.

    window.hour_of_day long Optional.

    Response:

    HTTP Status Code: 200

    Representation:

    {
        "value" : {
            "hour_of_day" : 1,
            "day_of_week" : "string"
        }
    }

    Response Type:

    Name Type Description
    bold = required
    value maintenance_window com.vmware.vmc.model.MaintenanceWindow.
    value.day_of_week string Possible values are:
    • DAY_OF_WEEK_SUNDAY
      • ,
    • DAY_OF_WEEK_MONDAY
      • ,
    • DAY_OF_WEEK_TUESDAY
      • ,
    • DAY_OF_WEEK_WEDNESDAY
      • ,
    • DAY_OF_WEEK_THURSDAY
      • ,
    • DAY_OF_WEEK_FRIDAY
      • ,
    • DAY_OF_WEEK_SATURDAY
      • ,

    Optional.

    value.hour_of_day long Optional.

    Errors:

    HTTP Status Code Type Description
    401 com.vmware.vapi.std.errors.Unauthenticated Unauthorized
    500 com.vmware.vapi.std.errors.ConcurrentChange Conflict with exiting reservation
    400 com.vmware.vapi.std.errors.InvalidRequest The reservation is not in a state that's valid for updates
    403 com.vmware.vapi.std.errors.Unauthorized Access not allowed to the operation for the current user
